# Contributing to CVG Neuron Public

Thank you for your interest in contributing! This repository showcases **AI-generated code samples** from CVG Neuron AI. We welcome contributions that help demonstrate the quality and utility of AI-generated code.

## How to Contribute

### 1. Share Your AI-Generated Code

Have you used CVG Neuron to generate useful code? Share it!

**Steps:**
1. Use CVG Neuron at [neuron.cleargeo.tech](https://neuron.cleargeo.tech) to generate code
2. Test the generated code in your workflow
3. Add header comments showing:
   - Query used to generate the code
   - AI generation notes
   - Training projects it was based on (if shown)
4. Submit a pull request

**Example Header:**
```python
"""
[Script Name]
Generated by: CVG Neuron AI
Based on: [Number] projects from CVG training data
Query: "[Your query to CVG Neuron]"
Use Case: [What problem this solves]
"""
```

### 2. Improve Documentation

Help others understand how to use these AI-generated samples:
- Add usage examples
- Improve code comments
- Create tutorials or guides
- Add screenshots or diagrams

### 3. Report Issues

Found a problem with an AI-generated sample?
- Open an issue describing the problem
- Include the sample file name
- Provide steps to reproduce
- Suggest improvements if possible

### 4. Request Examples

Need AI-generated code for a specific workflow?
- Open an issue with "Example Request" label
- Describe your use case
- Provide context (project type, tools, constraints)
- Community members may generate and share examples

## Contribution Guidelines

### Code Samples

**Must have:**
- ✅ Header comment showing AI generation source
- ✅ Clear documentation of purpose and use case
- ✅ Example usage section
