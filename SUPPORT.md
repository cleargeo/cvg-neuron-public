## ğŸš€ Before Opening an Issue

To help us help you faster, please:

### 1. Search Existing Resources
- Check [existing issues](https://github.com/cleargeo/cvg-neuron-public/issues)
- Review [discussions](https://github.com/cleargeo/cvg-neuron-public/discussions)
- Consult the [documentation](https://clearviewgeographic.com/docs)
- Check the [FAQ](https://clearviewgeographic.com/faq)

### 2. Gather Information
Prepare the following details:
- **CVG Neuron version:** Check with `--version` flag
- **Operating System:** OS name and version
- **Shell:** bash, zsh, sh, etc.
- **Error messages:** Full error output
- **Steps to reproduce:** Detailed reproduction steps

### 3. Create a Minimal Example
If possible, provide:
- Minimal code that reproduces the issue
- Sample data (if applicable)
- Configuration files (sanitized)

## ğŸ› ï¸ Troubleshooting

### Common Issues

#### Installation Problems
1. Verify system requirements
2. Check file permissions
3. Ensure dependencies are installed
4. Review installation logs

#### Configuration Issues
1. Validate configuration syntax
2. Check environment variables
3. Verify file paths
4. Review access permissions

#### Runtime Errors
1. Check CVG Neuron version compatibility
2. Review error logs
3. Verify input data format
4. Test with minimal example

### Debug Mode

Enable verbose logging:
```bash
export CVG_DEBUG=1
./your-script.sh
```

### Log Locations

- **Linux/Mac:** `~/.cvg-neuron/logs/`
- **Windows:** `%APPDATA%\CVG-Neuron\logs\`
